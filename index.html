Projetos em destaque

Vaultwarden DevOps (Docker, AWS EKS, CI/CD, Terraform, ArgoCD)

Repositório voltado à implementação de uma aplicação real (Vaultwarden) com pipeline completo de provisionamento e deploy em ambiente cloud. Provisionamento de infraestrutura na AWS utilizando Terraform, automatizado via GitHub Actions, incluindo configuração de recursos e integração com EKS. Deploy da aplicação em Kubernetes (EKS) utilizando ArgoCD, seguindo abordagem GitOps para gerenciamento declarativo e versionado dos manifests. Exposição da aplicação via Ingress (Traefik) com TLS automatizado (cert-manager/Let’s Encrypt), incluindo configuração de DNS e acesso seguro externo. Implementação de persistência com volumes do Kubernetes integrados à AWS, garantindo armazenamento de dados para aplicação stateful.

Projeto Intensivo de Containers e Kubernetes (PICK)

Criei uma pipeline completa de entrega de aplicações containerizadas, combinando construção de imagens seguras e minimalistas com apko e melange, assinatura e validação de integridade com Cosign e Kyverno, e deploy automatizado em Kubernetes via CI/CD. Além disso, incorporei práticas essenciais de SRE, como observabilidade com kube-prometheus-stack (Prometheus, Grafana e Alertmanager) e testes de carga com Locust, garantindo não apenas a entrega contínua, mas também segurança, monitoramento e validação de performance em todo o ciclo da aplicação. Pipeline: Código da aplicação Build de pacotes (melange) Build da imagem (apko) Assinatura (Cosign) Push para registry Deploy no Kubernetes Validação por políticas (Kyverno) Monitoramento (Prometheus/Grafana) Teste de carga (Locust)

Observabilidade corporativa e administração em escala

Desenvolvi o projeto combinando monitoramento, resposta a incidentes, automação e gestão de infraestrutura. Implantação e evolução do monitoramento com Zabbix, Grafana e Prometheus, estruturando visibilidade sobre servidores, links, serviços e componentes críticos. Desenvolvimento de templates, alertas, mapas de topologia e integrações para comunicação automatizada, melhorando detecção proativa de falhas e acompanhamento operacional. Uso de Ansible, GPO e scripts em Python para gerenciamento e controle em massa de aproximadamente 500 hosts, demonstrando capacidade de automação, padronização e operação em escala. Projeto com fundamentos de observabilidade, automação operacional, troubleshooting e confiabilidade.

Alertas críticos via chamadas telefônicas com Zabbix + Python + Asterisk

Projeto autoral voltado a automação de incidentes e redução de tempo de resposta operacional. Integração entre Zabbix, Python e Asterisk para transformar eventos críticos em chamadas telefônicas para serviços essenciais. Aplicado a cenários como disponibilidade de servidores, status de sistemas, disponibilidade de links e temperatura da sala de servidores. Demonstra capacidade de integrar monitoramento com automação prática, agregando valor real para ambientes que exigem resposta rápida a indisponibilidades.

Monitoramento do IX.br com Python, Zabbix e Grafana

Desenvolvi o projeto para coleta automatizada, integração com observabilidade e descoberta dinâmica de itens. Script em Python realiza web scraping da página de status do IX.br, consolida cada local/serviço em valor binário e entrega JSON para coleta pelo Zabbix. O desenho do projeto inclui item mestre, LLD (Low-Level Discovery), criação dinâmica de itens e triggers por serviço monitorado, demonstrando entendimento prático do ecossistema Zabbix. O projeto também contempla visualização em dash
